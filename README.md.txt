# The Last of Us Game
***

This project is a tribute to "The Last of Us," created as part of an Object-Oriented Programming (OOP) course. Developed using Java and JavaFX, it aims to capture the captivating narrative and immersive world of the original masterpiece.

## Introduction
"The Last of Us" captivated gamers worldwide with its emotional storytelling and intense gameplay. This project aims to reflect some of that essence while demonstrating the principles of object-oriented design and programming.

## Features
- **Object-Oriented Design:** Adheres to OOP principles, showcasing encapsulation, inheritance, and polymorphism.
- **JavaFX GUI:** Provides an intuitive graphical user interface for an engaging experience.
- **Inspired Gameplay:** While not a direct clone, it draws inspiration from the mechanics and atmosphere of "The Last of Us."
